# Support

Thanks for using Shroud (`shroud-ai`). Here is how to get help.

## Questions & usage help

- Read the [README](./README.md) — install, quick start, the wrapper examples, and the API.
- Search existing [GitHub issues](../../issues) — your question may already be answered.
- Open a new issue with the **question** label if it isn't.

## Bug reports

Open a GitHub issue with:

- the version of `shroud-ai`,
- your Node version and OS,
- a **minimal reproduction** (the smallest input + code that shows the problem),
- what you expected and what happened.

## Security issues

Do **not** open a public issue. Follow [SECURITY.md](./SECURITY.md).

## Response expectations

Shroud is maintained by a small team on a best-effort basis. There is no paid support tier
and no SLA. Clear, reproducible reports are handled fastest.
